# Expense Tracker

Веб-приложение для учёта личных расходов. Позволяет добавлять траты по категориям, отслеживать общую сумму расходов и анализировать затраты по каждой категории. Все данные сохраняются в браузере через localStorage и не пропадают после перезагрузки страницы.

## Что работает

* Добавление новой траты с категорией, суммой, датой и описанием
* Автоматический подсчёт общей суммы расходов
* Автоматический подсчёт расходов по категориям
* Удаление расходов из списка
* Сохранение данных в localStorage
* Восстановление данных после перезагрузки страницы
* Отображение пустого состояния при отсутствии расходов

## Что не доделано

* Редактирование существующих расходов
* Фильтрация расходов по периоду
* Экспорт данных в CSV
* Визуализация данных в виде диаграмм

## Что я узнал во время работы

Во время выполнения проекта я научился работать с localStorage для хранения данных между перезагрузками страницы. Разобрался с сериализацией объектов через JSON.stringify() и восстановлением данных через JSON.parse().

Также закрепил навыки работы с DOM: создание элементов через JavaScript, обработка событий форм и динамическое обновление интерфейса без перезагрузки страницы.

Дополнительно изучил обработку ошибок при чтении данных из localStorage с помощью конструкции try...catch, что позволяет избежать падения приложения при повреждённых данных.
